📋 如何取得設定？ 1. 前往 console.firebase.google.com
2. 選擇您的專案 → ⚙️ 專案設定
3. 向下捲到「您的應用程式」→ 選擇 Web 應用程式
4. 複製 firebaseConfig 物件或貼上完整 JSON / SDK 設定碼
